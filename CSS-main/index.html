Satır İçi CSS (inline)

CSS kodu uygulanacak bir paragraf - inline CSS
buradaki html komtularının css kodu diretk p tagının içinden geliyor

Sayfa İçi CSS (internal)

CSS kodu uygulanacak bir paragraf - internal CSS
buradaki html komutlarının css kodu head tagındaki style dan geliyor.

Sayfa İçi CSS (external)

CSS kodu uygulanacak bir paragraf - external CSS
bu sayfadaki html komutlarının css kodu style.css dosyasından geliyor.
